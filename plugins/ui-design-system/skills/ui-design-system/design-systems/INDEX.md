# Design Systems Catalog — Index

74 vendored design systems organized by category. Each entry points at a
`<system>/DESIGN.md` file with a 9-section spec (visual theme, color palette,
typography, layout, components, motion, imagery, voice, edge cases) extracted
from real production CSS.

Use this catalog from `ui-design-system` **curated mode** — pick a system that
matches the desired vibe, copy its tokens into `docs/design/MASTER.md`, and
adapt to the project's domain and stack.

For attribution and licensing, see [NOTICE.md](NOTICE.md).

## Picking a system

Match by **product type and vibe**, not industry:

- Building a developer SaaS dashboard? Linear, Vercel, Cursor, Supabase.
- Building a consumer marketplace? Airbnb, Pinterest, Shopify, Notion.
- Building an AI / LLM product? Claude, Cohere, Mistral, Ollama (per desired warmth).
- Building a fintech tool? Stripe is the reference — adopt its restraint, then layer your brand color over its skeleton.
- Want bold / brutalist? x.ai, Lamborghini, Wired.
- Want soft / friendly? Cohere, Lovable, Notion, Cal.
- Want premium-corporate? Apple, Stripe, BMW, IBM Carbon.

When in doubt, **pick two systems** with contrasting strengths and tell the
generator to take typography discipline from one and color discipline from the
other. The 9-section format makes mix-and-match cheap.
